# ChargeBatt: Cargador compacto de Baterías Li-ion (1S)

Este proyecto presenta el diseño de una **PCB compacta** para un cargador de baterías de litio de una sola celda (1S).

---

## Características Principales

* **Diseño Compacto:** La PCB mide **menos de 2 cm por lado**, lo que la hace ideal para dispositivos portátiles y espacios reducidos.
* **Capacidad de Carga:** Soporta una corriente de carga máxima de hasta **2.1 A**.
* **Configuración:** La corriente de carga se ajusta mediante la selección de **resistencias de *sensing*** ubicadas en la capa inferior (*Bottom Layer*) de la PCB, permitiendo flexibilidad en la configuración.

---

## Nota de Implementación

Este cargador está diseñado exclusivamente para baterías de **una celda (1S)** de ion de litio o polímero de litio.
